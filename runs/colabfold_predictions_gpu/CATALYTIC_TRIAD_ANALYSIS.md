# Catalytic Triad Geometry Analysis

Quantitative analysis of Ser-Asp-His catalytic triad geometry for WT and all candidates.

## Key Metrics

- **Ser OG ↔ His N distance**: Should be ~2.5-3.5 Å for H-bond
- **His N ↔ Asp O distance**: Should be ~2.6-3.2 Å for functional triad
  - **> 4.0 Å**: Non-functional (Asp too far)
  - **2.6-3.2 Å**: Functional (good distance)
  - **3.2-4.0 Å**: Marginal (may need optimization)
  - **< 2.6 Å**: Too close (unlikely)
